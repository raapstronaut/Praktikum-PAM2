## Fitur Utama
Aplikasi **News Feed Simulator** adalah simulasi feed berita yang terus bertambah otomatis.
1. Feed berita bertambah otomatis setiap 2 detik sekali
2. Bisa filtering berita berdasarkan kategori
3. Berita ditampilkan dalam format yang rapi (judul + info kategori)
4. Ada fitur Read Count
5. Saat berita diclick, aplikasi menampilkan detail beritanya (diambil secara async/simulasi)

## Cara Menjalankan (Android Studio)
1. Clone / download repository:
   - `https://github.com/raapstronaut/Praktikum-PAM.git`
2. Buka folder **Prak2** menggunakan Android Studio:
   - **File → Open → pilih folder `Prak2/`**
3. Tunggu proses **Gradle Sync** sampai selesai.
4. Jalankan aplikasi dengan menekan tombol **Run**.
5. Pilih emulator/device Android, lalu aplikasi akan terbuka dandan berita akan muncul otomatis
