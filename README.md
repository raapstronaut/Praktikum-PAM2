# Praktikum PAM

Repository ini isinya kumpulan tugas MK PAM (Pengembangan Aplikasi Mobile)

## Daftar Tugas
- **Prak2 — News Feed App**
  - Folder: [Prak2](./Prak2)

- **Prak3 — My Profile App**
  - Folder: [Prak3](./Prak3)

- **Prak4 — Develop Profile App**
  - Folder: [Prak4](./Prak4)

## Cara Membuka Project
- Buka **Prak2**: buka folder `Prak2/` di Android Studio (File → Open)
- Buka **Prak3**: buka folder `Prak3/` di Android Studio (File → Open)
- Buka **Prak4**: buka folder `Prak4/` di Android Studio (File → Open)
