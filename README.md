## Deskripsi
Pengembangan **Notes App** dengan **Platform Features** menggunakan:
- **Koin Dependency Injection**
- **DeviceInfo** dengan `expect/actual`
- **NetworkMonitor** dengan `expect/actual`

Pada tugas ini, aplikasi menampilkan **informasi device** pada halaman **Settings** dan **indikator status jaringan** pada halaman utama.

## Fitur Utama
- Menggunakan **Koin Dependency Injection** untuk mengelola dependency aplikasi
- Menampilkan **Device Info** pada halaman **Settings**
- Menampilkan **Network Status Indicator** pada halaman utama
- Menggunakan `expect/actual` untuk implementasi fitur platform-specific
- `DeviceInfo` digunakan untuk mengambil informasi perangkat
- `NetworkMonitor` digunakan untuk memantau kondisi jaringan online/offline

## Cara Menjalankan (Android Studio)
1. Pilih branch **week-8**
2. Clone / download repository:
   - `https://github.com/raapstronaut/Praktikum-PAM.git`
3. Buka folder project tugas praktikum 8 menggunakan Android Studio.
4. Tunggu proses **Gradle Sync** sampai selesai.
5. Jalankan aplikasi dengan menekan tombol **Run**.
6. Pilih emulator/device Android, lalu aplikasi akan terbuka.
